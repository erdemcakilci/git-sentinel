# Security Policy

## Supported Versions

We actively provide security patches for the following versions of **Git-Sentinel**:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | 🚀 Yes             |
| < 1.0   | ❌ No              |

## Reporting a Vulnerability

We take the security of Git-Sentinel seriously. If you find a security vulnerability, please do not open a public issue. Doing so exposes users to potential exploits before we can patch them.

Instead, please report vulnerabilities by email to:
**Erdem Çakılcı** - [erdemcakilci@gmail.com](mailto:erdemcakilci@gmail.com)

Please include:
1. A detailed description of the vulnerability.
2. Steps to reproduce the issue (proof of concept code or configuration).
3. The potential impact of the exploit.

We will acknowledge receipt of your report within 48 hours and work with you to patch the issue and coordinate a public release containing the fix. Thank you for helping keep our users safe!
